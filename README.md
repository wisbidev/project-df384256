# aiteam Landing Page

Single-page SaaS-style landing page for **aiteam**, an AI-powered platform that automatically builds software via Telegram.

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Deployment**: Vercel

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.

### Backend

Backend services are located in the `backend/` directory. Refer to `backend/README.md` for setup instructions.

## Design

- **Dark mode only** — no light mode toggle
- **Color palette**:
  - Background: `#0B1121`
  - Accent: `#3B82F6`
  - Card: `#1E2938`
  - Muted: `#94A3B8`
  - Heading: `#F8FAFC`
- **Mobile-first responsive** design
